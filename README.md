# TaskFlow

TaskFlow es una aplicación web para la gestión de tareas personales. Permite a cada usuario crear, editar, eliminar, buscar, filtrar y ordenar sus tareas. Incluye un dashboard de estadísticas básicas de productividad.

## Objetivo
Facilitar la organización personal mediante una interfaz intuitiva que permita administrar el ciclo completo de una tarea desde su creación hasta su finalización.

## Stack tecnológico
- Frontend: React + TypeScript
- Backend: Node.js + TypeScript
- Base de datos: PostgreSQL
- Autenticación: JWT
- Contenerización: Docker
